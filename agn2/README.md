# AGN2

`agn2/` contains governance metadata and local control-plane source for
AGNProtocol.

## Contents

| Path | Purpose |
|---|---|
| `SYSTEM_IDENTITY.md` | System role summary |
| `SYSTEM_LAYOUT.md` | Runtime layer map |
| `system_manifest.json` | Portable system metadata |
| `governance/` | Constitution and policy configuration |
| `control_plane/` | Tauri control-plane source |
| `admin_profile.example.json` | Local operator profile template |

Local operator profile files are ignored by git.

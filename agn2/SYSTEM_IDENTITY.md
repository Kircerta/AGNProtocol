# System Identity

AGNProtocol is a local-first governed agent runtime.

## Roles

| Role | Responsibility |
|---|---|
| Operator | Starts tasks, reviews output, approves high-risk actions |
| Control plane | Displays state and submits governed commands |
| Governance layer | Applies policy, review, and emergency-stop checks |
| Runtime layer | Executes bounded provider, memory, review, vision, and desktop work |

The runtime records evidence and follows governance decisions.

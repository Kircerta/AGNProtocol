# Memory Recall Before Action Overlay

Use this overlay when the task depends on prior decisions, preferences, operating constraints, or repeated failure patterns.

## What This Overlay Adds

- recall before execution
- avoid rediscovering old constraints
- preserve reusable learnings as durable memory

## Default Questions

1. What durable decisions already constrain this area?
2. What broke last time?
3. What preferences or conventions should shape this work?
4. What result should be preserved as operating memory afterward?
